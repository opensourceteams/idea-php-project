#  mac idea中开发php


###  安装 brew

```aidl
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

```


###  安装php 

```aidl
brew install php 

```


###  安装后php 安装在目录
- 安装目录

```aidl
 /usr/local/Cellar/php/7.3.3
 
```
- 查看php 版本
```aidl
 /usr/local/Cellar/php/7.3.3/bin/php -v

```


### a.php
```aidl
<!DOCTYPE html>
<html>
<body>

<?php
echo "Hello World!";
?>

</body>
</html>


```
